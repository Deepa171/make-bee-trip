# Make Bee Trip 

A Travel & Tourism landing page built using React JS and Node JS.

## Features
- Flight Booking (API Integration)
- Bus Booking (Mock API)
- Train Booking (Mock API)
- Hotel Booking (Manual + API)

## Tech Stack
- React JS
- Node JS
- Express
- Bootstrap

## Project Structure
- frontend/
- backend/

## Run Locally

### Backend
cd backend
node server.js

### Frontend
cd frontend
npm start
